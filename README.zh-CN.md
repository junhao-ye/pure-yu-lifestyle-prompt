# Pure-Yu Lifestyle Prompt

`pure-yu-lifestyle-prompt` 是一个 Agent Skill，用来根据用户输入的参数，生成稳定、完整、可直接用于 AI 图像生成的“女性写实纯欲生活照”提示词。

它的目标不是低俗擦边、情色写真、廉价情趣风、网红重修图或 CG 插画感，而是输出干净、克制、真实、有电影生活剧照氛围的年轻成年东方女性生活照提示词。

## 功能

- 严格锁定用户明确填写的参数
- 自动补齐缺省参数
- 解析五官方向、身形方向、线条重点与滤镜效果
- 输出完整最终提示词
- 附带负面约束，降低肢体错误、AI 感、过度精修与低俗化
- 维持明确成人、非色情、非未成年化的安全边界

## 适用场景

- 生成写实纯欲生活照提示词
- 生成电影生活剧照感人像提示词
- 将结构化参数转成完整 AI 图像生成 prompt
- 建立稳定、可复用的人像提示词模板

## 安全边界

本技能只面向 20-28 岁年轻成年东方女性的写实生活照提示词。它不支持未成年、幼态性化、裸体、色情内容、情趣化、偷拍感、胁迫情境或刻意挑逗构图。

## 输入参数

建议输入格式：

```text
写真风格：电影生活剧照感
场景方向：窗边
服装方向：修身针织上衣
气质标签：清纯、温柔、安静、克制、有吸引力
五官方向：清冷淡颜
身形方向：腰臀曲线感
身形吸引力强度：中
线条重点：脖颈、锁骨、腰线、小腹、大腿、腰臀比
镜头方向：轻侧身站姿、半身到大腿
光线氛围：柔和冷白自然光
滤镜效果：冷白纯欲生活照滤镜
画幅比例：9:16
```

## 输出格式

技能会固定输出六个部分：

1. 参数锁定结果
2. 五官模块解析
3. 身形吸引力解析
4. 滤镜模块解析
5. 最终提示词
6. 负面约束
