# Pure-Yu Lifestyle Prompt

`pure-yu-lifestyle-prompt` は、ユーザーが入力したパラメータをもとに、AI 画像生成で使える安定したプロンプトを作成する Agent Skill です。対象は、写実的で清潔感があり、抑制された雰囲気の成人東アジア女性のライフスタイルポートレートです。

このスキルは、低俗な刺激、露骨な成人向け表現、安っぽいランジェリー広告風、過度なインフルエンサー風レタッチ、CG やイラスト調を目的にしていません。日常写真のリアリティと、映画の生活シーンのような静かな空気感を重視します。

## 機能

- ユーザーが明示したパラメータを厳密に保持
- 未指定のパラメータを安全なデフォルトで補完
- 顔立ち、体型の方向性、ラインの重点、フィルター効果を解析
- 画像生成に使いやすい最終プロンプトを出力
- 破綻した人体、AI 感、過度な修正、低俗化を避けるネガティブ制約を付与
- 成人であり、非露骨・非未成年化の安全境界を維持

## 用途

- 写実的なピュア系ライフスタイル写真プロンプトの生成
- 映画的な日常ポートレート用プロンプトの生成
- 構造化されたパラメータを完全な画像生成 prompt に変換
- 再利用しやすい人物プロンプトテンプレートの作成

## 安全境界

このスキルは、20-28 歳の若い成人東アジア女性を対象とした写実的なライフスタイル写真プロンプトのみを扱います。未成年、未成年風の性的表現、ヌード、露骨な性的内容、フェティッシュ化、盗撮風、強制的な状況、挑発的な構図には対応しません。

## 入力例

推奨フォーマット：

```text
写真スタイル：映画的な日常スチル感
シーン：窓辺
服装：フィット感のあるニットトップ
雰囲気タグ：清楚、やさしい、静か、抑制的、魅力的
顔立ちの方向：クールで淡い顔立ち
体型の方向：ウエストとヒップの曲線感
身体的魅力の強度：中
ラインの重点：首、鎖骨、ウエストライン、下腹部、太もも、ウエストとヒップの比率
カメラ方向：軽い横向きの立ち姿、上半身から太ももまで
光：柔らかいクールホワイトの自然光
フィルター：クールホワイトのピュア系ライフスタイル写真フィルター
アスペクト比：9:16
```

## 出力形式

スキルは常に次の 6 セクションを出力します：

1. パラメータ固定結果
2. 顔立ちモジュール解析
3. 身体的魅力解析
4. フィルターモジュール解析
5. 最終プロンプト
6. ネガティブ制約
