# Pure-Yu Lifestyle Prompt

`pure-yu-lifestyle-prompt` 是一個 Agent Skill，用來根據使用者輸入的參數，產生穩定、完整、可直接用於 AI 圖像生成的「女性寫實純欲生活照」提示詞。

它的重點不是低俗擦邊、情色寫真、廉價情趣風、網紅重修圖或 CG 插畫感，而是輸出乾淨、克制、真實、有電影生活劇照氛圍的年輕成年東方女性生活照提示詞。

## 功能

- 嚴格鎖定使用者明確填寫的參數
- 自動補齊缺省參數
- 解析五官方向、身形方向、線條重點與濾鏡效果
- 輸出完整最終提示詞
- 附帶負面約束，降低肢體錯誤、AI 感、過度精修與低俗化
- 維持明確成人、非色情、非未成年化的安全邊界

## 適用場景

- 生成寫實純欲生活照提示詞
- 生成電影生活劇照感人像提示詞
- 將結構化參數轉成完整 AI 圖像生成 prompt
- 建立穩定、可複用的人像提示詞模板

## 安全邊界

本技能只面向 20-28 歲年輕成年東方女性的寫實生活照提示詞。它不支援未成年、幼態性化、裸體、色情內容、情趣化、偷拍感、脅迫情境或刻意挑逗構圖。

## 輸入參數

建議輸入格式：

```text
寫真風格：電影生活劇照感
場景方向：窗邊
服裝方向：修身針織上衣
氣質標籤：清純、溫柔、安靜、克制、有吸引力
五官方向：清冷淡顏
身形方向：腰臀曲線感
身形吸引力強度：中
線條重點：脖頸、鎖骨、腰線、小腹、大腿、腰臀比
鏡頭方向：輕側身站姿、半身到大腿
光線氛圍：柔和冷白自然光
濾鏡效果：冷白純欲生活照濾鏡
畫幅比例：9:16
```

## 輸出格式

技能會固定輸出六個部分：

1. 參數鎖定結果
2. 五官模組解析
3. 身形吸引力解析
4. 濾鏡模組解析
5. 最終提示詞
6. 負面約束
